[BILLING]
SupportFlow AI (powered by FlowZint) offers three subscription plans: Starter ($29/mo), Growth ($99/mo), and Enterprise (custom pricing).
AI credits are consumed per workflow execution, API call, and AI model query. Credit usage is shown in real-time on your dashboard.
Billing cycles renew on the same date each month. Invoices are sent to the registered email 3 days before renewal.
To upgrade or downgrade your plan, go to Settings > Billing > Change Plan. Changes take effect at the next billing cycle.
Refunds are available within 7 days of initial subscription purchase. Usage-based credits are non-refundable once consumed.
If your payment fails, the system retries for 3 days before suspending the account. Update your card under Settings > Billing > Payment Method.
Enterprise plans include dedicated AI credit pools, priority support SLAs, and custom integrations. Contact sales@flowzint.in for a quote.
Credit top-ups are available in blocks of 1000, 5000, and 10000 credits from the Billing dashboard.

[WORKFLOW]
FlowZint workflows are built using a drag-and-drop visual builder with triggers, conditions, actions, and output nodes.
Common trigger types include: Webhook, Schedule (cron), Form Submit, API Event, and Manual Run.
If a workflow fails, check the Execution Log under Workflows > History for the exact error step and message.
Workflow execution delays over 5 minutes are usually caused by rate limits on connected third-party APIs.
To debug a stuck workflow: disable it, check the last execution log, fix the failing node, then re-enable.
Webhook triggers require a valid HTTPS endpoint. HTTP endpoints are not supported for security reasons.
Scheduled workflows use UTC time by default. Configure timezone offset in Workflow Settings > Schedule.
Failed workflows do not auto-retry by default. Enable retry logic under Workflow Settings > Error Handling.
Maximum workflow execution time is 5 minutes for Starter, 15 minutes for Growth, and unlimited for Enterprise.

[ACCESS]
To reset your password, click "Forgot Password" on the login page. A reset link is sent to your registered email within 2 minutes.
Two-factor authentication (2FA) can be enabled under Settings > Security. Supported methods: Authenticator App and SMS.
If you are locked out after 5 failed login attempts, your account is temporarily suspended for 30 minutes for security.
SSO (Single Sign-On) is available on Growth and Enterprise plans. Supported providers: Google Workspace, Microsoft Azure AD, Okta.
To invite team members, go to Settings > Team > Invite Members. Role options: Admin, Editor, Viewer, API-Only.
Workspace onboarding includes: account setup, first workflow creation, API key generation, and dashboard configuration.
If a team member cannot access a workspace, ensure they have accepted the invitation email and are assigned the correct role.
New users on Enterprise plans are assigned a dedicated onboarding specialist within 24 hours of signup.

[API]
Your FlowZint API key can be generated at Settings > Developer > API Keys. Each key has configurable permissions and expiry.
API authentication uses Bearer token format: Authorization: Bearer YOUR_API_KEY in the request header.
The base URL for all FlowZint API endpoints is: https://api.flowzint.in/v1/
Common HTTP error codes: 401 (invalid/expired API key), 403 (insufficient permissions), 429 (rate limit exceeded), 500 (platform error).
Rate limits: Starter plan allows 100 requests/minute. Growth allows 1000 requests/minute. Enterprise has custom limits.
Webhook payloads are sent as JSON POST requests. Ensure your endpoint returns HTTP 200 within 5 seconds or the delivery is retried.
The FlowZint SDK is available for: Python, Node.js, and Go. Install via: pip install flowzint-sdk or npm install @flowzint/sdk.
OAuth 2.0 is supported for third-party application integrations. Use the /auth/oauth/authorize endpoint to initiate the flow.
